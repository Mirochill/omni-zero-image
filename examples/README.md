# Examples

`prompts.jsonl` contains prompts used by the smoke tests and benchmark plan.

Real output images are intentionally not committed until they are generated by
the repository code in a verified run. Draft outputs can be generated with:

```bash
omnizero generate --mode draft --prompt "a coherent futuristic city at sunrise" --out outputs/city.png
```

Model-quality examples require a trained checkpoint and should include the
exact commit, checkpoint hash, command, seed, hardware, and runtime.
