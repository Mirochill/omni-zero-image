## Minimum Evidence Before Quality Claims

Do not claim frontier quality until all of these are true:

- CI passes on the public repository.
- A checkpoint hash is published.
- The checkpoint is generated by documented training/distillation commands.
- Text-to-image examples include command, prompt, seed, hardware, and runtime.
- Image-to-image examples include source image, mask when used, prompt, seed,
  hardware, and runtime.
- Automated evaluation covers prompt adherence, image quality, text rendering,
  identity/reference preservation, mask locality, and latency.
- Human evaluation compares against named baselines on the same prompt set.

## Suggested Evaluation Set

- 50 photorealistic product prompts.
- 50 portrait/character consistency prompts.
- 50 complex scene prompts.
- 50 typography/design prompts.
- 50 image-edit prompts with masks.
- 50 reference-style transfer prompts.
